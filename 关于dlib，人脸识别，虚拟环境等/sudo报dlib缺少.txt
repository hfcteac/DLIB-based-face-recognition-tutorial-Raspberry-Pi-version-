## 解决dlib缺失问题

在树莓派终端中运行以下命令安装dlib和其它必要组件：

````bash
# 首先激活您的虚拟环境
source ~/face_env/bin/activate

# 安装依赖
sudo apt-get update
sudo apt-get install -y cmake libopenblas-dev

# 安装dlib和imutils
pip install dlib imutils

# 如果编译安装太慢，可以尝试预编译版本（根据您的Python版本选择）
# pip install https://github.com/jloh02/dlib/releases/download/v19.22/dlib-19.22.99-cp39-cp39-linux_aarch64.whl
````
