
# 在树莓派上安装 dlib 库

这个错误表明你需要在当前环境中安装 dlib 库。在树莓派上安装 dlib 可能比较耗时，因为需要编译。

## 方法一：在系统环境中安装 dlib（推荐）

```bash
# 安装编译依赖
sudo apt-get update
sudo apt-get install -y build-essential cmake libopenblas-dev liblapack-dev libx11-dev libgtk-3-dev
sudo apt-get install -y python3-dev python3-setuptools

# 安装 dlib
sudo pip3 install dlib
```

## 方法二：在虚拟环境中安装并运行

如果你已经创建了虚拟环境，则需要在虚拟环境中安装 dlib 并使用特殊方法以 sudo 权限运行：

```bash
# 激活虚拟环境
source ~/face_env/bin/activate

# 安装 dlib（可能需要 30 分钟到 1 小时完成）
pip install dlib

# 使用 sudo 运行虚拟环境中的 Python
sudo ~/face_env/bin/python /home/pi/Desktop/face/eye_detect.py
```

## 方法三：使用简化版本（无需 dlib）

如果安装 dlib 太困难，可以考虑使用 OpenCV 的替代功能来实现类似功能，修改代码以移除 dlib 依赖。

注意：dlib 在树莓派上编译可能需要 30 分钟到数小时，请耐心等待。如果内存不足，可以增加交换空间：

```bash
sudo dphys-swapfile swapoff
sudo nano /etc/dphys-swapfile
# 修改 CONF_SWAPSIZE=2048
sudo dphys-swapfile setup
sudo dphys-swapfile swapon
```
